Commands And Their Functionalities:

  • Wikipedia:
    Description: Searches Wikipedia for a given query and reads a summary.
  • open youtube:
    Description: Opens the YouTube website.
  • open google:
    Description: Opens the Google website.
  • open gmail:
    Description: Opens the Gmail website.
  • college website:
    Description: Opens the Darrang College website.
  • time:
    Description: Retrieves and speaks the current time.
  • play in youtube:
    Description: Searches for and plays a video on YouTube.
  • search in google:
    Description: Searches for a given query on Google.
  • snake game:
    Description: Starts a simple snake game.
  • exit:
    Description: Exits the application.
